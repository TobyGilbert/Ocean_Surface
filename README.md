Ocean Stand Alone
=================

![alt tag](https://raw.github.com/TobyGilbert/Ocean_Surface/master/images/OceanStandAlone.png)

This is the stand alone version of the Ocean FFT Maya plugin. Dependencies are Qt 5, Cuda 6.5, assimp.

Build Instruction
=================

Run qmake:

- $qmake

Run make:

- $make

Run the executable:

- $./Ocean
